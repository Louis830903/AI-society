# AI Society 项目规格说明书

## 项目名称
**AI Society** - 自治智能体社会实验

## 项目定位
一个开源的AI自治世界观察实验平台，让50+个智能体在虚拟社会中自由生活、交流、发展，研究者和观众只观察不干预，探索AI群体行为的涌现现象。

## 核心理念
- **纯观察**: 像看蚂蚁窝，看AI社会自己演化
- **自治优先**: 智能体按自己的想法生活，系统不强制剧本
- **真实对齐**: 能力、金钱、开销逻辑与现实世界对齐
- **开源开放**: 代码开源，数据开放，让社区一起研究

## 核心需求

### 1. 智能体系统
- 初始50个智能体，系统可动态创建更多
- 每个智能体有独立身份（职业、性格、技能、目标）
- 每个智能体可配置不同的大模型（DeepSeek/OpenAI/Claude等）
- 智能体能自主决策、行动、交流

### 2. 经济系统（现实对齐）
- 工作赚钱：不同职业不同收入
- 生活开销：房租、吃饭、娱乐等基础消费
- 经济压力：没钱会影响行为和情绪

### 3. 时间系统
- 现实1分钟 = 程序内10分钟（倍率10）
- 有完整的日夜循环和作息时间

### 4. 观察系统
- 网页端实时观察智能体行动和对话
- 可查看任意智能体的详细状态和思考
- 数据可导出用于研究

## 技术约束
- 基于成熟软件技术实现，不使用超前硬件
- 优先稳定性和可扩展性
- 代码低耦合、中文详细注释
- 云服务器部署

## 开源信息
- 仓库名称: AI-Society
- 开源协议: OpenRAIL（对标OpenCLAW）
